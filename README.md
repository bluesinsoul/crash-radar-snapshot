# Crash Radar Feed

Machine-written daily snapshot of **SHORT / AVOID** signals over the broad top-~1000 crypto universe,
distilled from the local `CoinResearch/GemRadar` pipeline (CrashRadar lens: CoinGecko market+fundamentals +
DeFiLlama protocol fees/TVL/revenue/holders-revenue, over the `longshot_screener` price/volume backbone).
Consumed by the **Crash Radar** cloud routine, which can't reach those APIs directly.

Raw signals only — **no composite score**. Overvaluation Gates / Deterioration Divergences / Distribution
Traps; `short_watch` = confluence (>=2 aligned layers). `null` = NO DATA (not zero). Overwritten daily.
This is an over-valued / over-extended / deteriorating detector, **not** a contract-level rug scanner.
